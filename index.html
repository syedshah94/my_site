About


A little bit about myself
Hi, I'm Syed, a designer and full stack web developer in NYC. After four years of electrical engineering, I am pursuing a career more involved in design and programming. I enjoy creating fun and useful apps in my spare time. Right now, I'm working on mastering React-Native and want to start building mobile apps, so stay tuned for that!

Values
I enjoy working with colors and creating clean, modern web designs to ensure users have a fluid experience.
My goal is to use intuitive web design to create apps that help people out in their daily lives, to aid them with finding a parking spot or simply catching up on memes. I strive to create dynamic websites which make use of current and emerging technologies.

Skills
On the tech side, I implement the MVC design pattern for my apps. I prefer React for my front end design. I typically implement backend with either Rails or Express. For version control, I use git. And I also enjoy working with Bootstrap and Bulma.io for when I want to create a design quickly.
